# 🔐 Password Strength Checker

A Python-based password strength checker that evaluates passwords using common cybersecurity best practices.

## Features

- Minimum length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Password strength scoring
- Improvement suggestions

## Technologies

- Python 3
- Regular Expressions (re)

## Example

Input

Password123

Output

Strength: Medium

Suggestions:
- Add a special character.

## Scoring

| Score | Strength |
|--------|-----------|
|0-2|Weak|
|3-4|Medium|
|5|Strong|
