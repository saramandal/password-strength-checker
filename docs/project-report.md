# Project Report

## Objective

To evaluate password strength using cybersecurity best practices.

## Methodology

The application evaluates:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Each successful criterion increases the security score.

## Result

Passwords are classified into:

- Weak
- Medium
- Strong

Users receive recommendations to improve password security.

## Learning Outcomes

- Python programming
- Input validation
- Regular expressions
- Password security principles
